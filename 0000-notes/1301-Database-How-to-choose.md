|          | Structured data        | Semi-structured data | Unstructured data                                            |
| -------- | ---------------------- | -------------------- | ------------------------------------------------------------ |
| Data pie | 10 to 15 %             | 5% to 10%            | 80% to 90%                                                   |
| Example  | RDS table, spreadsheet | HTML, JSON, XML      | Multimedia (photo, audio, video), text document, scanned PDF |
